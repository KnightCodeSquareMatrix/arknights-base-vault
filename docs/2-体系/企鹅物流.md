# 企鹅物流

## 一句话结论

企鹅物流有两套**互斥**的贸易站路线——同一个德克萨斯不能同时走两条路。精 0 德克萨斯走「恩怨」（与拉普兰德搭配 +65% 效率），精 2 德克萨斯走「默契」（与能天使搭配）。另一个平行选项是不用德克萨斯，直接用能天使 + 蕾缪安的「相伴」组合。两套路线效率相近但搭配完全不同。

## 两条路线

### 路线 A：德狼（德克萨斯精 0 + 拉普兰德）

| 干员 | 练度 | 作用 |
|------|------|------|
| 德克萨斯 | **精 0** | 恩怨：与拉普兰德同站 +65% 效率，但心情消耗 +0.3/h |
| 拉普兰德 | 精 2 | 醉翁之意·β：德克萨斯同行时订单上限 +4 |

德克萨斯的「恩怨」只在精 0 时存在。精 1 后二级技能替换「恩怨」变成其他效果，**精 2 后「恩怨」消失、变为「默契」**。所以德狼路线要求德克萨斯**保持精 0**。效率面板 0.65（65%）。

拉普兰德精 2 后的醉翁之意·β 提供 +4 订单上限。合计 65% 效率 + 上限 +4。配合第三散件（如裁缝 β），三人组可逼近一线水平。

### 路线 B：能蕾（能天使精 2 + 蕾缪安精 2）

| 干员 | 练度 | 作用 |
|------|------|------|
| 能天使 | 精 2 | 物流专家：+35% 纯效率；与蕾缪安同行额外 +25% |
| 蕾缪安 | 精 2 | 相伴：+20% 效率，与能天使同行额外 +25% |

能天使自身 35%，同行后蕾缪安追加 25%，合计能天使 60% + 蕾缪安 45% = 面板 1.05。再搭配第三散件（如裁缝 β）即可成形。

### 路线 C：能蕾 + 新能天使

| 干员 | 练度 | 作用 |
|------|------|------|
| 新约能天使 | 精 2 | 替代能天使的变体，基础面板略低 |

效率 0.6 起步，不如标准能蕾组合但门槛更低（不需要老能天使）。

## 德克萨斯精 0 vs 精 2 的关键区别

这是整个游戏中最特殊的干员成长路径之一：

- **精 0**：持有「恩怨」（与拉普兰德 +65%）——走德狼路线
- **精 2**：持有「默契」（与能天使搭配）——走企鹅物流另一条路线

**二者不可兼得。** 德克萨斯的精 2 是德狼路线的终点——一旦精 2，「恩怨」永久消失。反过来，能蕾路线不需要德克萨斯，是完全独立的平行方案。

## 缺人降级

| 缺谁 | 方案 |
|------|------|
| 德狼无拉普兰德 | 不开 |
| 德狼无德克萨斯精 0 | 无法走德狼，只能看能蕾 |
| 能蕾无能天使 | 新约能天使替补（约 0.6 面板） |
| 能蕾无蕾缪安 | 效率打折扣，不建议 |
| 两条路都缺核心 | 用日常散件替代 |

## 与其他体系的关系

- 企鹅物流不与任何中枢体系冲突（不需要中枢干员）
- 与巫恋核、但书链在 243 双贸易站下共存——可以占第二个贸易站
- 德狼路线不抢精 2 资源——德克萨斯只需要精 0
