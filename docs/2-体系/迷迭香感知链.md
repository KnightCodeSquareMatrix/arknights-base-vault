# 迷迭香感知链

## 一句话结论

迷迭香感知链是一套跨设施全局资源协作体系。迷迭香（制造站）和黑键（贸易站）绑定同上同下，以「感知信息」为核心中间件——中枢的夕产感知、宿舍的爱丽丝/车尔尼产感知、办公室的絮雨产感知——全部汇入全局池后转化为迷迭香的制造效率和黑键的贸易效率。在 243/153/333 三电站布局下，满配约 **80–90% 制造效率 / 40–45% 贸易效率**。

## 何时值得开

### 硬性条件

- 三电站布局（243 / 153 / 333）。二电站布局下宿舍等级降低，爱丽丝/车尔尼产出缩水，但仍勉强可用
- 迷迭香 **必须精 2**
- 黑键 **必须精 2**
- 迷迭香和黑键同上同下（一起上班、一起下班）

### 不开的情况

- 迷迭香或黑键任意一个缺失 → 不开
- 无絮雨，且缺少夕 / 令 / 爱丽丝 / 车尔尼等其他感知来源 → 感知来源太少，不建议开
- 恢复班 → 不启用

### 中枢取舍：八幡海铃 vs 琴柳（二选一）

办公室的絮雨需要中枢琴柳提升人力联络效率以维持感知产出，但中枢只有 5 个席位。当八幡海铃（叙拉古链）也需要中枢位时，产生二选一冲突：

| 分支 | 中枢选择 | 办公室搭配 | 取舍 |
|------|---------|----------|------|
| **琴柳 + 絮雨** | 琴柳占中枢 1 席 | 絮雨在办公室 → 感知信息 | 制造效率优先；办公室锁定絮雨 |
| **八幡海铃 / 焰狐龙梓兰** | 八幡海铃占中枢 1 席（叙拉古链） | 办公室选高效率散件（斥罪/凯尔希思衡托/焰狐龙梓兰） | 办公室产出优先；感知链降档（缺絮雨感知源） |

**判断依据**：玩家更看重制造效率 → 选琴柳+絮雨+迷迭香；更看重办公室公招刷新 → 选八幡海铃/焰狐龙梓兰。

> **焰狐龙梓兰**：办公室联络速度 +10%，是公招刷新取向的散件补位；她不生产感知信息。

## 成套组成

### 核心双人（缺一不可）

| 干员 | 设施 | 练度 | 作用 |
|------|------|------|------|
| 迷迭香 | 任意制造站 | 精 2 | 超感产感知 → 意识实体/念力转化为制造效率 |
| 黑键 | 非巫恋贸易站 | 精 2 | 乐感 → 无声共鸣 → 贸易效率 |

### 感知生产者（越多越好）

| 干员 | 设施 | 练度 | 产出 |
|------|------|------|------|
| 夕 | 中枢 | 精 0 即可 | 心情 >12 时产感知 10；心情 <12 时转产人间烟火 |
| 絮雨 | 办公室 | 精 2（推荐） | 记忆碎片 → 感知（Lv.3 办公室约 +20） |
| 爱丽丝 | 宿舍 | 精 2 | 梦境呓语：每宿舍等级 +1 感知 |
| 车尔尼 | 宿舍 | 精 2 | 琴键漫步：每宿舍等级 +1 感知 |
| 令 | 中枢 | 精 2 | 替代夕，需保持心情 ≤12 |
| 琴柳 | 中枢 | 精 2 | 可选项，与絮雨搭配时补齐感知 |

琴柳对感知体系本身无直接贡献——她的作用是给絮雨提高办公室人力效率，间接维持感知产出。

## 缺人降级路径

从外围往核心砍：

```
满配（迷迭香 90% 制造 / 黑键 45% 贸易）
  ↓ 砍令（心情控制）
档 1（80% / 40%）
  ↓ 砍宿舍（爱丽丝 + 车尔尼）
档 2（75% / 37.5%）
  ↓ 砍夕
档 3（~55% / ~27.5%）
  ↓ 缺迷迭香或黑键
关闭
```

| 档位 | 保留 | 砍掉 | 迷迭香效率 | 黑键效率 |
|------|------|------|----------|---------|
| 满配 | 核心 + 令(≤12) + 夕 + 爱丽丝 + 车尔尼 + 絮雨 + 琴柳 | — | 90% | 45% |
| 档 1 | 核心 + 夕 + 爱丽丝 + 车尔尼 + 絮雨 | 令 | 80% | 40% |
| 档 2 | 核心 + 夕 + 絮雨 | 宿舍二人 | 75% | 37.5% |
| 档 3 | 核心 + 絮雨 | 夕 | ~55% | ~27.5% |
| 关闭 | — | 缺核心 | — | — |

## 贸易站搭配

黑键所在的贸易站优先级：

1. **最佳**：但书 + 黑键 + 高效散件——但书站拿最高效率散件，黑键做第三人
2. **次选**：可露希尔 + 黑键 + 次高效散件
3. **不可**：巫恋 + 黑键（不能同站）

## 与各体系关系

| 体系 | 关系 |
|------|------|
| 但书叙拉古链 | 完全兼容，黑键 + 但书同站是最优搭法 |
| 自动化组 | 完全互补，243 下各占一条赤金线 |
| 巫恋裁缝核 | 不与黑键同站 |
| 人间烟火 | 第二班替代场景：夕心情 <12 后转产人间烟火 |
| 喀兰中枢（灵知） | 中枢位互斥 |

## 班次与房间约束

- 迷迭香 + 黑键上 2 班休 1 班（班 1 + 班 2；班 3 休息），日均 18h
- 夕在中枢随体系进出（班 1 + 班 2；班 3 休息，中枢位让出）
- 黑键固定在非巫恋贸易站，迷迭香固定在任意制造站
- 夕在中枢（精 0 即可），令可替代（需严格控心情 ≤12）
- 爱丽丝 + 车尔尼尽可能放同一宿舍
- 絮雨在办公室（仅 Lv.3 办公室有产出），必须与迷迭香同班次

## 数值锚点

### 迷迭香制造效率

| 配置 | 效率 |
|------|------|
| 满配（含令控心情） | **90%** |
| 档 1（无令） | **80%** |
| 档 2（无宿舍） | **75%** |
| 仅核心 | ~55% |

### 黑键贸易效率

| 配置 | 效率 |
|------|------|
| 满配 | **45%** |
| 档 1 | **40%** |
| 档 2 | **37.5%** |
| 仅核心 | ~27.5% |

相对普通散件（制造 35–40%），满配迷迭香的制造端净提升 50 个百分点以上。

## 明确不选清单

- 缺迷迭香精 2 或黑键精 2 → 不开
- 无絮雨，且缺少夕 / 令 / 爱丽丝 / 车尔尼等其他感知来源 → 感知源不足，不开
- 巫恋已占黑键候选贸易站且无其他空贸易站 → 不开（黑键不与巫恋同站）
- 恢复班 → 不启用
- 四电站非常规布局 → 未经验证，不建议
- 办公室选桑葚（人间烟火组）替代絮雨 → 产出较低，使用优先级低于絮雨感知信息
