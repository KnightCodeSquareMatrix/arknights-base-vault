# 深海链（阿戈尔中枢）

## 一句话结论

深海猎人体系是**中枢→制造全加成体系**：歌蕾蒂娅 E2 在中枢为每名进驻制造站的深海猎人提供 +10% 全制造效率（单站上限 90%）。斯卡蒂/幽灵鲨/安哲拉/乌尔比安四名自身无基础制造技能的干员，靠歌蕾蒂娅 buff 成为高额制造效率人，且可在 0 心情/红脸状态下工作。代价是歌蕾蒂娅心情消耗极快（~3.2/h），单班最长 **7.5h**。

## 何时值得开

### 硬性条件

| 条件 | 说明 |
|------|------|
| 歌蕾蒂娅 | **必须 E2**（集群狩猎·β） |
| 斯卡蒂 + 幽灵鲨 + 安哲拉 + 乌尔比安 | 至少 3 人进制造，4 人全有最佳 |
| 制造站 | ≥2 间（否则无法有效分散深海猎人） |
| 产物 | **全制造通用**（赤金/经验均可） |
| 班次 | 高峰班短班（歌蕾蒂娅极限 7.5h） |

### 不开的情况

- **无歌蕾蒂娅 E2** → 不开
- **深海猎人制造干员 < 3** → 不开（收益太低）
- **歌蕾蒂娅心情不足以支撑 ≥1 个班次** → 不开
- **自动化组已占深海候选制造站且无其他空站** → 互斥，不开

## 成套组成

### 架构

```
控制中枢：歌蕾蒂娅 E2
  ├── 集群狩猎·β → 每站外深海猎人 +10% 制造（单站 ≤90%）
  └── 潮汐守望 → 歌蕾蒂娅自身心情消耗 ~3.2/h

制造站分配（243 例，2+2 最优）：
  ┌─ 赤金/经验站 A: 乌尔比安 + 斯卡蒂（2 人，2×4×10% = 80%）
  └─ 赤金/经验站 B: 幽灵鲨 + 安哲拉（2 人，2×4×10% = 80%）
```

### 成员

| 角色 | 干员 | 设施 | 最低练度 | 作用 |
|------|------|------|---------|------|
| **体系核心** | 歌蕾蒂娅 | 中枢 | E2 | 集群狩猎·β 提供制造加成 + 潮汐守望心情消耗 |
| 制造受 buff 者 | 乌尔比安 | 制造站 | E0（无需技能） | 吃站外深海计数，单人可成站 |
| 制造受 buff 者 | 斯卡蒂 | 制造站 | E0（无需技能） | 同上 |
| 制造受 buff 者 | 幽灵鲨 | 制造站 | E0（无需技能） | 同上 |
| 制造受 buff 者 | 安哲拉 | 制造站 | E0（无需技能） | 同上 |

**关键特性**：四名制造干员**自身无基础制造技能**，效率全部来自中枢歌蕾蒂娅 buff。因此：
- 他们对制造产物类型无偏好（赤金/经验均可）
- **不可脱离歌蕾蒂娅独立工作**——歌蕾蒂娅心情耗尽则 buff 消失，四人效率归零。必须同上同下
- 他们的练度只影响是否占用 `tier_up` 资源，不影响制造效率

## 效率公式

**公式**：`min(本站深海人数 × 全基建制造站中深海猎人总数 × 10%, 90%)`

即：每个深海猎人的加成 = 本站深海人数 × 全局（所有制造站中的）深海猎人数 × 10%，单站上限 90%。

| 深海猎人配置 | 布局 | 全制造人数 | 各站效率 | 总加成 |
|------------|------|----------|---------|--------|
| **最优 2+2** | 243 | 4 | 80% / 80% | **160%** |
| **最优 1+1+2** | 333 | 4 | 40% / 40% / 80% | **160%** |
| 1+3（封顶浪费） | 243 | 4 | 40% / 90% | **130%** |
| 4 人同站（封顶浪费） | 243 | 4 | 90% | **90%** |
| 仅 2 人同站 | 243 | 2 | 40% | **40%** |
| 仅 1 人 | 243 | 1 | 10% | **10%（不值得）** |

**最优分配策略**：将深海猎人**分散到尽可能多的制造站**：
- 243（2 制造站）：**2+2**（每站 80%，合计 160%）
- 333/153（≥3 制造站）：**1+1+2** 分配
- 避免 3+1 和全同站（封顶浪费人数）

## 缺人降级路径

```
满配（4 深海 + 歌蕾蒂娅 E2，2+2 分两站 160%）
  │ 缺乌尔比安
  ├─ 档 1（3 深海：斯卡蒂+幽灵鲨+安哲拉 同站 90%）
  │ 缺乌尔比安+安哲拉
  ├─ 档 2（2 深海同站 40% → 收益过低，不建议）
  │ 缺歌蕾蒂娅 E2 或 <2 深海
  └─ 关闭
```

| 档位 | 深海人数 | 分配 | 总加成 |
|------|---------|------|--------|
| 满配 | 4 | 2+2 或 1+1+2 | 160% |
| 档 1 | 3 | 2+1 | ~90%+ |
| 档 2 | 2 | 同站 | 40% |
| 关闭 | <2 | — | — |

**缺人替补原则**：
- 4 人全时：2+2（243）或 1+1+2（333）
- 剩 3 人时：2+1 分配，避免三人同站封顶
- 剩 2 人时：放同一站（40%）优于分两站（各 20%）
- 剩 1 人时：仅 10%，不值得占中枢位

## 与其他体系的共存与互斥

### 互斥

| 组合 | 规则 |
|------|------|
| 集群狩猎·β × 自动化组（温蒂/森蚺/清流） | ❌ 同站不可叠加，**清零效果优先生效** |
| 深海猎人 × 自动化同站 | 深海猎人加成被清零，避免同站 |
| 龙门中枢（斩业星熊+诗怀雅） | 中枢位紧张。龙门组 2 席 + 歌蕾蒂娅 1 席 + 夕/海铃等，5 位可能不够 |

### 兼容

| 体系 | 关系 |
|------|------|
| 迷迭香感知链 | **完全兼容**。不同制造站互不影响 |
| 红松林经验 | **完全正交**。经验/赤金分配自由 |
| 但书叙拉古链 | **正交**。不影响贸易 |
| 巫恋裁缝核 | **正交** |

> 深海链**不进入** `meta_chain` 互斥组，与但书/喀兰/推王/怪猎可共存。

## 班次与房间约束

### 心情核算

| 心情项目 | 数值 | 来源 |
|---------|------|------|
| 基础消耗 | 1.0/h | 默认 |
| 潮汐守望（5 猎人宿舍外） | +2.5/h | 5 × 0.5 |
| 中枢满人全局心情减免 | -0.25/h | 中枢全满 buff |
| 集群狩猎中枢心情恢复 | -0.05/h | 技能自带 |
| **合计** | **3.2/h** | |
| **最长单班** | **~7.5h** | 24 心情→耗尽 |

### 房间分配

| 房间 | 约束 |
|------|------|
| 控制中枢 | 歌蕾蒂娅 E2（1 席），满 5 人全局心情减免 |
| 制造站 A | 乌尔比安 + 斯卡蒂（80%） |
| 制造站 B | 幽灵鲨 + 安哲拉（80%） |
| 制造站产物 | 赤金/经验均可，无偏好 |

### 班次轮换——同上同下

- **必须同进同出**：歌蕾蒂娅 + 全体制造深海猎人在同一班次
- **班次节奏**：建议使用短班（~7h 一班），或歌蕾蒂娅上早班/晚班部分时段
- **休息班时**：制造站退散件，中枢换散件（阿米娅/凯尔希等）
- 深海猎人不需要宿舍回心情（红脸工作也无所谓）

## 明确不选清单

1. 无歌蕾蒂娅 E2 → 不开
2. 深海猎人制造干员 < 3 → 不开
3. 自动化组已占深海候选制造站且无其他空站 → 互斥，不开
4. 无法支持 ≤7.5h 短班 → 歌蕾蒂娅心情不够，不开
5. 恢复班 → 不启用成套
