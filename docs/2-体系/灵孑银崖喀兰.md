# 灵孑银崖喀兰

## 一句话结论

灵知（中枢）降低谢拉格干员效率、增加订单上限 → 孑精 1+ 读被压低的效率、压缩更少的上限 → 银灰 +4 订单上限有更大空间发挥。满配等效约 **192% 贸易效率**。这是一个需要完整搭配才能启动的体系——缺灵知不开，孑精 0 和精 1 走的是完全不同的两条路。

## 何时值得开

- 灵知 **必须精 2**（精密计算 → 每名谢拉格干员 −15% 效率 +6 上限）
- 孑 **必须精 1+**（市井之道）——精 0 孑不走这条路线
- 银灰精 2（+20% 效率 +4 上限）
- 第四个位置放喀兰工具人：崖心 / 讯使 / 锏 / 琳琅诗怀雅
- 贸易站 Lv.3

## 两套孑路线

| 配置 | 孑练度 | 机制 | 适用场景 |
|------|--------|------|---------|
| 灵孑银崖 | 精 1+ | 市井之道压缩上限 + 灵知减效控制 | 有灵知中枢 |
| 孑拉德 | 精 0 | 摊贩经济 per-gap +4%，不压缩上限 | 无灵知中枢，配合德狼 |

没有灵知时，精 1 孑的市井之道会基于完整的他人效率来压缩上限——银灰的 +4 被压掉大半，组合收益亏损。此时精 0 孑反而更实用。

## 成套组成

| 干员 | 设施 | 练度 | 作用 |
|------|------|------|------|
| 灵知 | 中枢 | 精 2 | 精密计算：每名谢拉格干员 −15% 效率 +6 上限 |
| 孑 | 贸易站 | 精 1+ | 市井之道：每 10% 他人效率压缩 1 上限 |
| 银灰 | 贸易站 | 精 2 | 喀兰之主：+20% 效率 +4 上限 |
| 崖心 / 讯使 / 锏 / 琳琅诗怀雅 | 贸易站 | 精 2 | 谢拉格/喀兰工具人 |

## 效率基准

| 配置 | 等效贸易效率 |
|------|-----------|
| 灵知中枢 + 孑 + 银灰 + 崖心 (精 2) | **等效 192%** |
| 灵知中枢 + 银灰 (45%) + 崖心 (40%) | 等效 75%（单看效率面板） |
| 无灵知、有银灰 + 崖心 | 显著下降 |

银灰在 Lv.3 站自身 +20% 效率 +4 上限，灵知的 −15% 效率在固定加成之后、孑读取之前生效——孑读到的是银灰 +20% − 灵知 15% = 净 +5%，比不压上限时候少了很多。

## 缺人降级

- 缺灵知精 2 → 不开。退回到孑拉德路线（精 0 孑 + 德狗 + 拉狗）
- 缺银灰 → 不开，少了 +4 上限体系没有意义
- 缺喀兰工具人 → 找房内有谢拉格/喀兰 tag 的干员替补

## 与其他体系的关系

- 与但书叙拉古链、推王龙门、怪猎中枢互斥（中枢 meta_chain 互斥组，四选一）
- 与巫恋核不冲突，不同贸易站
- 与红松林兼容（中枢位不同，红松林用焰尾+薇薇安娜）

## 数值锚点

- 满配等效：**192%** 贸易效率（孑 + 银灰 + 崖心 + 灵知中枢）
- 无灵知：不开
- 银灰面板：+20% 效率 +4 上限，灵知注入后净效率降低、上限空间增大
