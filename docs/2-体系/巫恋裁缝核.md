# 巫恋裁缝核

## 一句话结论

巫恋（精 2）将同房其他干员效率归零，每人为自身 +45% 效率；龙舌兰（精 2）提供投资技能增加订单报酬；第三人为裁缝 β 持有者（卡夫卡/柏喙/明椒/折光，精 1 只有裁缝α，精二升级为裁缝β），将 4 赤金订单概率推至 85%。满配 **138% 贸易效率 + 46% 赤金效率**。三人缺一不可，推荐全部精 2。

## 何时值得开

- 巫恋精 2 + 龙舌兰精 2 + 任意裁缝技能 持有者（α也可）
- 贸易站必须 Lv.3
- 裁缝技能需要连续在岗达到峰值：裁缝 α 约 3 小时，裁缝 β 约 5 小时
- 不与但书链冲突——243 双贸易站下各占一站

## 效率分档

| 组合 | 贸易效率 | 赤金效率 | 说明 |
|------|---------|---------|------|
| 巫恋II + 龙舌兰II + 裁缝β | **138%** | **46%** | 满配，裁缝 β 峰值 |
| 巫恋II + 龙舌兰II + 裁缝α | 129% | 38% | 裁缝 α 峰值，差了 9 个百分点 |
| 巫恋II + 龙舌兰II + 白板 | 124% | 33% | 亏 14 个百分点——不要这样放 |
| 巫恋II + 龙舌兰0 + 白板 | 108% | 17% | 龙舌兰没精 2 直接腰斩 |
| 巫恋II + 裁缝β + 白板 | 93% | — | 缺龙舌兰，收益急剧下降 |

> 注：巫恋II + 龙舌兰II + 但书 = 178% 贸 + 33.5% 金，但但书和可露希尔不能同站——这个组合只有在不触发可露希尔订单时成立，实际排班中不常用。

## 成套组成

| 干员 | 练度 | 作用 |
|------|------|------|
| 巫恋 | **精 2** | 低语：归零同房他人效率，每人为自身 +45% |
| 龙舌兰 | **精 2** | 投资·β：订单报酬 +500 龙门币 |
| 第三人 | **精 2** | 裁缝 β 持有者：卡夫卡 / 柏喙 / 明椒 / 折光 |

裁缝 α 精 1 解锁，精 2 升级为裁缝 β。四人都五星干员。选谁只看你的box、养成材料。

## 为什么第三人必须是裁缝 β

巫恋归零发生在裁缝技能改变订单类型之前。归零只清零干员自身的效率数字，裁缝的效果是提高 4 赤金订单概率——这个能力来自技能本身，不看效率值。所以裁缝 β 干员即使被归零到 0% 效率，照样能把 4 赤金概率推到 85%。

换成其他任何类型的干员——赤金虚拟产线（鸿雪、绮良）、纯效率干员——归零后效率消失，等于白占一个位置。

## 缺人降级

- 缺裁缝 β → 退而求其次用裁缝 α（损失 9 个百分点贸易效率）
- 缺龙舌兰精 2 → 不要开，白板龙舌兰只有 108% 贸，不如散件
- 缺巫恋精 2 → 不开

## 与但书链的关系

巫恋核和但书链在 243 布局下各占一个贸易站，不互斥。巫恋核的优先级（16）低于但书链（20），如果只有一个贸易站，但书链优先。

## 数值锚点

- 满配：138% 贸易效率 + 46% 赤金效率
- 裁缝 α 替代：129% 贸 + 38% 金
- 缺龙舌兰：108% 贸 + 17% 金，不推荐
