# 发电站机制

## 一句话

发电站提供电力支撑其他设施建造，进驻干员加速无人机充能。无人机可投入制造站/贸易站加速生产。虚拟电站是自动化组等体系的基石。

## 设施基础

| 等级 | 提供电力 | 进驻上限 |
|------|---------|---------|
| Lv.1 | 60 | 1 人 |
| Lv.2 | 130 | 1 人 |
| Lv.3 | 270 | 1 人 |

最多建造 3 座。降级时需保证总电力够支撑其他设施运行。

## 无人机机制

无人机是发电站的核心产出物，用于加速制造站和贸易站的生产。

### 充能基础规则

- **基础充能速度**：无干员进驻时，**6 分钟产生 1 架无人机**。
- **充能仅受发电站干员影响**：无人机的充能速度只取决于发电站进驻干员的加成，不受中枢光环、宿舍等其他设施影响。
- **充能无视上限持续进行**：即使当前无人机数量已达到持有上限，充能过程仍然继续。超出上限的部分直接丢弃。
- **发电站不会因满上限而停摆**：受上述机制影响，发电站进驻干员后始终处于工作状态——不存在"无人机满了就停工"的情况。这意味着发电站干员的心情会持续消耗，不会因为无人机溢出而减少消耗。

### 充能加速公式

进驻干员后，充能速度 = **6 ÷ (1 + 充能加成之和)** 分钟/架。

- **基础加成**：只要有人进驻，至少 +5%（即使进驻的干员没有充能技能）
- **技能加成**：+10% ~ +20%，多个发电站的加成各自独立、不叠加

常见充能干员：
- **承曦格雷伊**：充能速度 = (无人机上限 ÷ 10)%，非固定 +20%
- +20%：雷蛇·β、炎狱炎熔·γ、澄闪、天空盒、海霓·β
- +15%：伊芙利特、异客、格劳克斯·β、清流、深靛、布丁、阿消
- +10%：Castle-3 / Lancet-2 / THRM-EX / 正义骑士号等作业平台

### 无人机加速公式

每架无人机可**减少 3 分钟**制造站或贸易站的基础耗时（即干员效率加成之前的时间）。加速后无人机立即消耗，等效产出加成约 0.685 倍（公孙工具人表系数）。

例如：制造站基础耗时 72 秒的赤金配方，投入 1 架无人机后缩短为 72 − 180 = −108 秒（即实际减免 3 分钟基础耗时，具体减多少看配方的原始时长）。

## 虚拟电站

**承曦格雷伊（精 2）「晨曦」**：其他发电站没有作业平台进驻时，该发电站数量额外 +1。

这个 +1 不提供实际电力（不能用来建新设施），但会计入设施数量统计。对温蒂「仿生海龙」（每个发电站 +15% 制造生产力）来说，虚拟电站和真实电站一样计入——这就是自动化组必须带承曦格雷伊的原因。

## 干员技能的特殊联动

- **缪尔赛思「生态科主任」**：基础 +10%，基建内每有 1 名**莱茵生命阵营**干员（最多 5 名）额外 +3%，最高 +25%。注意：淬羽赫默和溯光星源是异格阵营、不属莱茵生命阵营——不计入缪尔赛思。缪尔赛思在莱茵科技体系中算双收益——自己的充能加成和阵营计数同时贡献。
- **空构「技术交流」**：时间爬升型，首小时 +10~15%，此后每小时 +1%，最终 +15~20%。
- **菲莱（精 2）「灵河共鸣」**：每间宿舍每级额外 +0.5%。满宿舍全满级下加成可观。
- **正义骑士号**：进驻后让野鬃所在制造站 +5% 生产力——罕见的发电站影响制造站的联动。
- **CONFESS-47**：有其他拉特兰干员同在发电站时 +5%。

## 发电站数量对布局的影响

- **153 布局**（1 贸易 + 5 制造 + 3 发电）：电站少 → 总充能慢 → 无人机少 → 贸易/制造加速少
- **243 布局**（2 贸易 + 4 制造 + 3 发电）：标准配置，电站数量够
- **333 布局**（3 贸易 + 3 制造 + 3 发电）：电站多充能快，但制造站少
- 虚拟电站不计入实际电力但对温蒂有用——243 下真实电站 3 + 虚拟 1 = 温蒂 +60% 制造生产力
